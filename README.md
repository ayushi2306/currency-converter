# 💱 Currency Converter

A Python-based Currency Converter project built step by step to demonstrate the progression from a basic command-line application to a feature-rich desktop application using live exchange rates.

Instead of creating separate repositories for each implementation, this repository showcases the complete learning journey. Each version introduces new concepts, libraries, and programming techniques while improving the functionality and user experience.

---

## 📌 Project Objective

The goal of this project is to understand how a real-world application evolves over time by gradually incorporating new technologies and features.

Throughout this project, different versions explore concepts such as:

* Python fundamentals
* Functions and modular programming
* Conditional statements
* Exception handling
* Dictionaries and data structures
* File handling
* GUI development with Tkinter
* Working with APIs
* JSON data parsing
* HTTP requests
* Input validation
* Professional project organization

## 🚀 Project Versions

### Version 1 – Basic Currency Converter

* Command-line interface
* Uses predefined exchange rates
* Focuses on Python fundamentals and conversion logic

---

### Version 2 – GUI Currency Converter

* Built using Tkinter
* User-friendly interface
* Interactive input fields and buttons
* Improved usability

---

### Version 3 – Live Currency Converter

* Fetches real-time exchange rates from an online API
* Introduces HTTP requests and JSON parsing
* Supports up-to-date currency conversions

---

### Version 4 – Advanced Currency Converter

* Enhanced user interface
* Better error handling
* Additional usability improvements
* More professional project structure
* Extra features and optimizations
